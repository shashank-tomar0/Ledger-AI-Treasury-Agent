Hardware-Enforced Agent Security Model

This dashboard acts as an interface layer. While the AI agent possesses the intelligence to monitor assets, calculate trade parameters, and generate payloads, it lacks the cryptographic authority to execute them. By decoupling the transaction preparation (handled by the agent CLI) from key custody (held within the Ledger Secure element), we ensure that a compromised agent cannot pivot, exceed treasury parameters, or fall victim to prompt injections without triggering a physical review on the device screen.
